# Food Ordering Analytics Dashboard

## Project Overview

This Tableau dashboard analyzes food ordering data to identify customer behavior, revenue trends, delivery performance, and popular cuisines.

## Dataset

Food ordering dataset containing:

- Customer Details
- City
- Cuisine
- Meal Type
- Order Value
- Rating
- Delivery Time

## Dashboard Features

- KPI Cards
- Revenue Trend
- Orders by City
- Cuisine Analysis
- Meal Type Analysis
- Customer Ratings
- Delivery Analysis

## Tools Used

- Tableau Public
- Microsoft Excel
